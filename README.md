🌐 Personal Profile Page
📌 Overview

This project is a simple personal profile webpage created using semantic HTML. It shows how to structure a webpage using meaningful tags.

🛠 Technologies Used
HTML5
🧠 Process
Created basic HTML structure using <!DOCTYPE html>, <html>, <head>, and <body>
Used <header> to display name and role
Organized content using <main> and <section>
Added an About section with a short introduction
Created a Skills section using <ul> and <li>
Added Contact details with a LinkedIn link using <a>
✅ Features
Simple and clean structure
Proper use of semantic HTML
Easy to understand
